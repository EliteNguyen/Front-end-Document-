1.Hướng dẫn CSS (CSS Tutorial)

CSS là ngôn ngữ mô tả phong cách của tài liệu HTML. (CSS is a language that describes the style of an HTML document.)
CSS mô tả cách các phần tử HTML sẽ được hiển thị. (CSS describes how HTML elements should be displayed.)
Hướng dẫn này sẽ dạy bạn CSS từ cơ bản đến nâng cao. (This tutorial will teach you CSS from basic to advanced.)
