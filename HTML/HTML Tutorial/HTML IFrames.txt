1.HTML iframe

Một iframe được sử dụng để hiển thị một trang web trong một trang web. (An iframe is used to display a web page within a web page.)


2.Cú pháp khung nội tuyến (Iframe Syntax)

Một iframe HTML được xác định bằng thẻ <iframe>: (An HTML iframe is defined with the <iframe> tag:)

Thuộc tính src chỉ định URL (địa chỉ web) của trang khung nội tuyến. (The src attribute specifies the URL (web address) of the inline frame page.)

3.Khung nội tuyến - Đặt chiều cao và chiều rộng (Iframe - Set Height and Width)

Sử dụng các thuộc tính chiều cao và chiều rộng để chỉ định kích thước của iframe. (Use the height and width attributes to specify the size of the iframe.)
Chiều cao và chiều rộng được chỉ định bằng pixel theo mặc định: (The height and width are specified in pixels by default:)

Khung nội tuyến - Xóa đường viền. (Iframe - Remove the Border)
Theo mặc định, iframe có viền xung quanh nó. (By default, an iframe has a border around it.)
Để xóa đường viền, thêm thuộc tính style và sử dụng thuộc tính viền CSS: (To remove the border, add the style attribute and use the CSS border property:)
