3. Tại sao học JavaScript? (Why Study JavaScript?)

JavaScript là một trong 3 ngôn ngữ mà tất cả các nhà phát triển web phải học:
(JavaScript is one of the 3 languages all web developers must learn:)

   1. HTML để xác định nội dung của các trang web
   (1. HTML to define the content of web pages)

   2. CSS để chỉ định bố cục của các trang web
   (2. CSS to specify the layout of web pages)

   3. JavaScript để lập trình hành vi của các trang web
   (3. JavaScript to program the behavior of web pages)

Các trang web không phải là nơi duy nhất sử dụng JavaScript. Nhiều chương trình máy tính để bàn và máy chủ sử dụng JavaScript. Node.js là nổi tiếng nhất. Một số cơ sở dữ liệu, như MongoDB và CouchDB, cũng sử dụng JavaScript làm ngôn ngữ lập trình của họ.
(Web pages are not the only place where JavaScript is used. Many desktop and server programs use JavaScript. Node.js is the best known. Some databases, like MongoDB and CouchDB, also use JavaScript as their programming language.)


4. Bạn có biết không? (Did You Know?)

JavaScript và Java là các ngôn ngữ hoàn toàn khác nhau, cả về khái niệm và thiết kế.
(JavaScript and Java are completely different languages, both in concept and design.)

JavaScript được Brendan Eich phát minh vào năm 1995 và trở thành tiêu chuẩn ECMA vào năm 1997. (Link)
(JavaScript was invented by Brendan Eich in 1995, and became an ECMA standard in 1997.)

ECMA-262 là tên chính thức của tiêu chuẩn. ECMAScript là tên chính thức của ngôn ngữ.
(ECMA-262 is the official name of the standard. ECMAScript is the official name of the language.)

Bạn có thể đọc thêm về các phiên bản JavaScript khác nhau trong chương Phiên bản JS. (Link)
(You can read more about the different JavaScript versions in the chapter JS Versions.)
